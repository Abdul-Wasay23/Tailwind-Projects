# Coastal Beef Fresh

## Overview
This repository contains the source code for the Coastal Beef Fresh website. Coastal Beef Fresh specializes in providing high-quality, pasture-raised beef products. The website is designed to showcase our premium offerings, categorized for easy shopping, and provide information about our sustainable sourcing and processing methods.

The website serves as a premium platform to inform customers about our quality standards, featured products, and to facilitate a seamless shopping experience across all devices.

## Features
- **Home Page**: Dynamic banner slider with progress indicators and featured products.
- **Shop By Categories**: Well-organized grid for browsing different beef product types (Steaks, Ground Beef, etc.).
- **New Arrivals**: Interactive product slider showcasing the latest available items.
- **How It Works**: Informative section detailing the process from sourcing to delivery.
- **Testimonials**: Customer feedback slider with a custom progress bar.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop (from 300px to 1500px+).

## Technologies Used
- **HTML5**
- **Tailwind CSS**: Modern utility-first CSS framework for rapid UI development.
- **JavaScript**: Interactive logic including slider initializations and mobile navigation.
- **Swiper.js**: High-performance touch-enabled mobile sliders.
- **Vite**: Modern frontend build tool for fast development and optimized production builds.

## Project Structure
- `index.html`: Main landing page.
- `src/css/custom.css`: Custom styles, theme variables, and Tailwind layers.
- `src/js/custom.js`: Interactive elements, Swiper initializations, and mobile menu logic.
- `tailwind.config.js`: Custom Tailwind CSS configuration including theme and screen breakpoints.
- `public/`: Static assets including fonts (Cutmark, Montserrat).
- `images/`: High-quality graphical assets.

## Usage
To view the website locally:
1. Clone this repository or download the source code.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run start
   ```
4. Open the provided local URL (usually `http://localhost:5173`) in your web browser.
